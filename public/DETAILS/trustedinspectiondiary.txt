   ```js

     build: {
       rollupOptions: {
         input: {
           main: 'index.html',
           about: 'about.html',
           services: 'services.html',
           contact: 'contact.html'
         }
       }
     }
   })
   ```

   This tells Vite to treat each HTML file as an entry point.
